# Sales Report Tool

売上CSVを自動で集計し、月別レポートを生成するPythonツールです。  
業務で発生しがちな「CSVの手作業集計によるミスや時間ロス」を減らすことを目的に作成しました。

## できること
- 複数の売上CSVを一括読み込み
- 月別に集計したレポートをExcelで出力
- 手作業での集計作業を自動化

inputフォルダに売上CSVを配置し、実行すると  
outputフォルダに月別レポートが生成されます。

## 使用技術
- Python
- pandas
- openpyxl

## 想定利用シーン
- 毎月の売上集計を手作業で行っている現場
- CSVベースのデータ整理業務
